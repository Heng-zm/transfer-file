How LocalDrop Works
Step 01
📤
Sender opens the Send tab and clicks "Create Offer" to generate a connection code.
Step 02
📨
Sender copies the offer code and sends it to the receiver (chat, email, etc).
Step 03
📥
Receiver pastes the code in the Receive tab and clicks "Generate Answer".
Step 04
🔗
Receiver sends back their answer code. Sender pastes it and connects.
Step 05
⚡
Direct P2P connection established. Drop files and transfer at full speed!
Bonus
🔒
100% encrypted WebRTC — no servers, no cloud storage, no size limits.
